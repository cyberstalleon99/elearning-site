How to Measure your Own Rib Length

This is a simple mathematical process. Determine the basket size that you prefer. For example, you want to make a 10" wide by 16" long by 6" deep basket. To get a length measurement, add the base length (16") and each side depth (6" on each side or 12"), and then add 6" (3" on each side) to tuck in at the top: 16" + 12" (2 sides) + 6" (tucking) = 34".

To get the width measurement, add the base width measurement, add the base width (10") and each side depth (6") with the same 6" for tucking: 10" + 12" (two sides) +6" = 28".

The only thing left to determine is the size of the reed to use, and therefore, how many of each to cut. Draw off the area and use a ruler to figure out how many pieces of each to use. Do not forget to add spaces between stakes. See the illustrations below.
